# 🛒 Boutique en Ligne — PHP & PostgreSQL

Application e-commerce full-stack développée dans le cadre d'un projet académique à l'ISG Bizerte.

---

## 📋 Description

Application web de vente en ligne avec deux interfaces distinctes :
- **Espace client** : navigation, panier, commande
- **Espace administrateur** : tableau de bord de gestion complet

---

## ✨ Fonctionnalités

### 👤 Côté Client
- Inscription et connexion sécurisée
- Catalogue de produits avec recherche
- Panier d'achat dynamique
- Passage de commande

### ⚙️ Côté Administrateur
- Tableau de bord (dashboard)
- Gestion des produits (ajout, modification, suppression)
- Suivi et gestion des commandes
- Administration des utilisateurs

---

## 🛠️ Technologies utilisées

| Technologie | Usage |
|-------------|-------|
| PHP | Back-end, logique métier |
| PostgreSQL | Base de données relationnelle |
| HTML/CSS | Structure et style des pages |
| JavaScript | Interactions front-end |

---

## 🗄️ Base de données

Base de données relationnelle conçue avec PostgreSQL :
- Modélisation entité-relation
- Gestion des utilisateurs, produits et commandes
- Contraintes d'intégrité et clés étrangères
